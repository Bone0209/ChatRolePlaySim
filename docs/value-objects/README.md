# 値オブジェクト仕様書インデックス

値オブジェクト（Value Object）は属性によって定義される不変オブジェクトです。

---

## 値オブジェクト一覧

| 値オブジェクト | 説明 |
|:---------------|:-----|
| [ParameterValue](./ParameterValue.md) | パラメータの値と可視性 |
| [Visibility](./Visibility.md) | 可視性（公開/非公開） |

---

## 設計原則

1. **不変性**: 一度生成したら変更不可。変更時は新インスタンス生成
2. **等価性**: 値が同じなら等しい（`equals()` メソッド）
3. **副作用なし**: メソッドは常に新しいオブジェクトを返す
