# ドメインエンティティ仕様書インデックス

ドメインエンティティはビジネスロジックの中核を担う、一意の識別子を持つオブジェクトです。

---

## エンティティ一覧

| エンティティ | 説明 |
|:-------------|:-----|
| [World](./World.md) | ゲームワールド |
| [GameEntity](./GameEntity.md) | プレイヤー/NPC |
| [ChatMessage](./ChatMessage.md) | チャットメッセージ |

---

## 設計原則

1. **不変性**: エンティティは基本的に不変。変更時は新しいインスタンスを生成
2. **ファクトリメソッド**: `create()` で新規作成、`reconstruct()` でDB復元
3. **カプセル化**: プライベートフィールドとgetterで状態を保護
4. **ビジネスメソッド**: ドメインロジックをエンティティ内に実装
