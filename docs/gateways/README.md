# ゲートウェイ仕様書インデックス

ゲートウェイは外部システムとの通信をカプセル化するインフラ層コンポーネントです。

---

## ゲートウェイ一覧

| ゲートウェイ | 説明 |
|:-------------|:-----|
| [LlmGateway](./LlmGateway.md) | LLM API通信 |

---

## 設計原則

1. **外部APIの隠蔽**: API詳細をインフラ層に閉じ込める
2. **設定の注入**: モデル設定は外部から注入可能
3. **ログ記録**: 通信内容を自動記録（デバッグ・監査用）
4. **エラーハンドリング**: タイムアウト、リトライ等を内部で処理
